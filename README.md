# ⚡ 极简 MC 启动器 (Simple Minecraft Launcher)

一个功能全面的 Minecraft 启动器，支持游戏下载、模组管理、服务器开服、内网穿透、光影安装、微软登录等。

## ✨ 功能特性

- 🎮 **游戏管理**：一键下载原版、Fabric、Forge、Quilt 等版本，自动配置
- 📦 **模组管理**：从 Modrinth 搜索、批量安装模组，支持版本指定，可禁用/启用模组
- 🖥️ **服务器开服**：支持 Paper / Spigot / Purpur / Fabric / Forge 服务端一键安装，内置控制台
- 🌐 **内网穿透**：集成 frp 客户端，一键配置并启动穿透
- 🎨 **光影包下载**：搜索并下载光影包到当前版本
- 🔐 **微软登录**：支持正版微软账号登录，自动刷新令牌
- 🧩 **整合包导入**：支持导入 .mrpack / .zip 格式整合包（Modrinth 格式自动下载模组）
- 🤖 **AI 助手**：调用 DeepSeek API，未知命令时智能回复（需自行配置 API Key  https://platform.deepseek.com/usage）
- 💾 **便携式**：所有数据存储在启动器同目录的 `pyLauncher` 文件夹，可随意移动
